# WSO2 Integrator Extension

A powerful VS Code extension for building integrations with WSO2 Integrator, combining visual design, low-code and pro-code development, and AI-assisted development experiences.

## Features

- **"Visual Integration Designer"**: Build integration flows quickly with an intuitive graphical interface
- **Low-Code + Pro-Code Parity**: Seamlessly switch between visual design and code for maximum flexibility
- **AI-Assisted Development**: Accelerate development with intelligent suggestions and interractive development with WSO2 Integrator Copilot
- **Local Development & Debugging**: Develop, run, test, and debug integrations directly within VS Code

## Installation

1. Install the WSO2 Integrator extension from the VS Code marketplace
2. Open the WSO2 Integrator activity bar icon to get started

## Extension Settings

This extension works with the settings of the underlying BI and MI extensions.
