# WSO2 Integrator Installation Guide

## Welcome to WSO2 Integrator!

This disk image contains the WSO2 Integrator installer package.

### Installation Instructions:

1. **Double-click the WSO2 Integrator installer** (`wso2-integrator-*.pkg`) to begin the installation process
2. Follow the on-screen prompts to complete the installation
3. The installer will set up all necessary components including:
   - WSO2 Integrator application
   - Ballerina runtime
   - Integration Control Plane (ICP) components

### System Requirements:

- **macOS**: 10.13 (High Sierra) or later
- **RAM**: Minimum 4GB (8GB recommended)
- **Disk Space**: At least 2GB of free space
- **Java**: Java 11 or later (if not bundled)

### Quick Start:

After installation:
1. Open Applications folder
2. Look for "WSO2 Integrator" in Applications
3. Double-click to launch the application

### Additional Resources:

- **Documentation**: https://wso2.com/integrator/docs/

### Troubleshooting:

If you encounter any issues during installation:
- Ensure you have administrator privileges
- Check that you have sufficient disk space
- Verify your macOS version is compatible
- Contact WSO2 support for additional assistance
